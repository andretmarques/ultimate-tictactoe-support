# Ultimate Tic-Tac-Toe — Support

Need help with **Ultimate Tic-Tac-Toe**, or found a bug? You're in the right place.

## Report a problem or ask for help

1. Go to the [**Issues**](https://github.com/andretmarques/ultimate-tictactoe-support/issues) tab.
2. Click **New issue**.
3. Describe what happened. If it's a bug, it helps to include:
   - Your device and OS version (e.g. iPhone 15, iOS 18).
   - The app version (shown on the app's menu / About screen).
   - Steps to reproduce the problem.
   - A screenshot, if relevant.

I read every issue and will respond as soon as I can.
